# 股票复盘笔记 MVP

## 运行

双击项目根目录下的 `start.bat`，脚本会自动编译、启动服务并打开浏览器。服务窗口会最小化，关闭那个窗口即可停止服务。

也可以手动运行：

```powershell
npm run dev
```

打开终端输出中的本地地址，默认是 `http://127.0.0.1:5173`。

## 测试

```powershell
npm run build
```

启动后可验证：

- 新建页面、编辑 Markdown，等待自动保存后刷新页面内容仍保留。
- 插入 `[[股票:贵州茅台]]` 形式的股票链接，并在右侧关联面板查看链接。
- 在渲染后的正文里点击股票链接，会按链接里的文字直接打开已有个股研究笔记；没有则自动创建一个空白个股研究笔记，不再匹配股票主数据。
- 交割单工作区可直接填写表格，股票名称和代码按输入原样保存。
